### Run the project
#### Backend
```sh
npm install
npm start
```
#### Frontend
```sh
npm install
npm run dev
```
Note: The site will be running on localhost:5173 or <your_local_ipv4>:5173.
To host on local network ipv4, change the url in vite.config to your ipv4.
